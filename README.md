# FakeGUI — WannaCry Prank Simulator

A Windows-only **prank simulator** that mimics the WannaCry ransom screen. It does **not** encrypt files or cause real damage.

## Features

- Fake Wana Decrypt0r 2.0 UI (scales to your screen size)
- Always-on-top window (close button does nothing)
- Downloads UI assets on first run into `pictures/`
- Optional wallpaper change and desktop folder prank after a few seconds

## Requirements

- Windows 10/11
- Python 3.10+

## Setup

```bash
pip install -r requirements.txt
python Wannacry.py
```

Assets are fetched automatically from [FakeGUI-Assets](https://github.com/Delexoo/FakeGUI-Assets) on first launch.

## Close the program

Type `123` in the Bitcoin address field and press **Enter**.
